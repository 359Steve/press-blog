# Joseph Joestar

努力做一个摆脱CV的前端工程师

> 好耶，你点开了这个页面，证明你想了解我更多，对吧？

嗨，我是 JoJo，一个喜欢写前端，也喜欢折腾各种项目的开发者。热衷把想法变成能动起来的页面，有时候是博客，有时候是一整个小系统。

我喜欢那些写着写着就突然 **「哦~原来可以这样!」** 的瞬间——那种卡了很久然后突然通了的感觉，比什么都上头。

这里记录一些我做过的东西，踩过的坑，还有偶尔的灵感闪现、奇怪尝试，以及那些让我哭笑不得的 bug。代码不一定完美，但每一段都算数。

有时候我也会写一些“不太技术”的东西，比如一个奇怪的交互、一个没什么用但很好玩的组件，或者一个“只是想试试”的项目——它们可能没什么实际意义，但挺有意思。

不写代码的时候，我偶尔会去钓会儿鱼。
坐在水边的时候，节奏会慢下来，脑子反而更清晰。有些问题是在电脑前想不明白的，但在盯着浮漂发呆的时候，答案会自己冒出来。

大概就是这样一个人：
一边写代码，一边踩坑，一边琢磨怎么把事情做得更有意思一点。

如果你刚好路过这里——
也许能看到你熟悉的坑，
或者某个瞬间，会让你也轻轻地说一句：
**「哦，原来还能这样。」**

## 自述

我最初只是想写博客。结果第一步就选了 `Nuxt`。功能很全，架构很完整，甚至有点……太完整了。后来我突然意识到：
我只是想写点东西，并不需要一套宇宙级解决方案。于是我换成了 `VitePress`。那一刻我才真正明白—— **“做网站”** 和 **“选择合适的技术”** 是两件事。现在这个博客更轻、更快，也更安静。
它不追求炫，只追求表达。

> 如果你也想使用它欢迎前去 [Github](https://github.com/359Steve/press-blog.git) 页面查看。

# 🎣 金沙洲新广佛大桥桥底

> 适合新手练手，白天和傍晚都比较容易找到钓位。

---

## 环境与特点

市区内难得的野钓点，据说巨物横行，鱼口好，有机会体验到与大鱼搏斗的刺激感。

## 注意事项

环境较差：有钓友反映桥底存在死鱼腐烂的恶臭问题，且暴晒，建议谨慎选择钓位。
